# web-apps

A collection of lightweight web applications served as static HTML files, backed by AWS API Gateway.

## Apps

| File | Description |
|------|-------------|
| `login.html` | User login |
| `register.html` | User registration |
| `dashboard.html` | Main dashboard |
| `admin.html` | Admin panel |
| `calculator.html` | Calculator |
| `todo-list.html` | To-do list |
| `prayer-times.html` | Prayer times |
| `electricity-tracker.html` | Electricity / KSEB tracker |
| `kseb.html` | KSEB utility |
| `utilities.html` | Utilities |

## Configuration

- `config.js` — Auto-generated by `deploy.sh`. Sets `window.API_BASE` pointing to the AWS API Gateway endpoint.
- `session-guard.js` — Handles session authentication across pages.

## Deployment
